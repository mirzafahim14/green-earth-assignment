## 1. What is the difference between var, let, and const?
- `var` is function-scoped and can be redeclared and reassigned.
- `let` is block-scoped and can be reassigned but not redeclared in the same scope.
- `const` is block-scoped and cannot be reassigned after declaration.

## 2. What is the difference between map(), forEach(), and filter()?
- `map()` creates a new array by transforming each element.
- `forEach()` only loops through the array and does not return a new array.
- `filter()` creates a new array with elements that match a condition.
